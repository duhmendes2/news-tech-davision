🚀 Instalação do Qwen AE Companion

⚠️ Importante: Esta extensão requer que você habilite extensões não assinadas no After Effects.

Passo 1: Habilitar Modo Debug

Windows:

  1. Feche o After Effects
  2. Pressione Win + R, digite regedit e pressione Enter
  3. Navegue até: HKEY_CURRENT_USER\SOFTWARE\Adobe\CSXS.10
  4. Clique com botão direito → Novo → Valor String
  5. Nomeie como PlayerDebugMode
  6. Defina o valor como 1

Mac:

  1. Feche o After Effects
  2. Abra o Terminal
  3. Execute: 
    defaults write com.adobe.CSXS.10 PlayerDebugMode 1

Passo 2: Copiar a Extensão

Windows:

Copie a pasta qwen-ae-companion para:

C:\Users\SEU_USUARIO\AppData\Roaming\Adobe\CEP\extensions\

Se a pasta CEP ou extensions não existir, crie-as.

Mac:

Copie a pasta qwen-ae-companion para:

/Users/SEU_USUARIO/Library/Application Support/Adobe/CEP/extensions/

Use Cmd + Shift + G no Finder para acessar a pasta Library.

Passo 3: Usar a Extensão

  1. Inicie o After Effects
  2. Vá em Window → Extensions → Qwen AE Companion
  3. Cole sua API Key do Qwen (obtenha em DashScope Console)
  4. Comece a criar! 🎨

📝 Exemplos do que Pedir

  • "Crie uma composição 1920x1080 com um texto animado"
  • "Adicione um efeito de blur e anime de 0 a 50"
  • "Crie 5 camadas de sólido com cores diferentes"
  • "Anime a posição criando um movimento de quicada"
  • "Aplique CC Sphere na camada selecionada"
⚠️ Controle Total: Esta extensão dá acesso completo ao After Effects via ExtendScript. O Qwen pode executar qualquer operação. Sempre revise o código antes de executar se estiver inseguro.

🆘 Problemas?

  • Verifique se o PlayerDebugMode está configurado corretamente
  • Certifique-se de que a pasta está em extensions (plural)
  • Reinicie o After Effects após instalar
  • Verifique se sua API Key do Qwen é válida

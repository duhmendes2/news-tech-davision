# 📊 Piloto 1: Learnings (Diego)

## ✅ O QUE DEU CERTO

### 1. **Voz Adam (11Labs)**
- ✓ Conexão excelente com persona "inspiracional/coach"
- ✓ Timbre natural, sem artificial
- **Padrão:** Usar Adam para personas masculinas + tons inspiracionais/educativos

### 2. **POV Smartphone Selfie Vertical**
- ✓ Autenticidade real
- ✓ Quebra padrão "vídeo profissional"
- ✓ Conecta melhor com público desconfiado de "fake"
- **Padrão:** Manter este POV para este tipo de conteúdo (depoimento, confissão, story pessoal)

### 3. **Características Generalistas da Imagem**
- ✓ Pessoa que se cuida (barba feita, cabelo organizado)
- ✓ Mas realista (não modelo comercial)
- ✓ Home office casual (hoodie cinza, background simples)
- ✓ Expressão confiante + humanizada
- **Padrão:** "Amateur authenticity" — pessoa REAL, não comercial

### 4. **⭐ WORKFLOW CRÍTICO: Você Narrou + Trocou Voz**
- **O que funcionou:** Você gravou o áudio pessoalmente com ORALIDADE NATURAL
- **Depois:** Copiou o texto para 11Labs + trocou sua voz por Adam
- **Resultado:** Oralidade natural (hesitações, ritmo) + voz profissional
- **Por que funciona:** Não depende do HeyGen sincronizar gestual — você aprovou a oralidade primeiro
- **🔴 MUDANÇA NA SKILL:** Fase 3 deve priorizar isto em vez de "HeyGen sync"

---

## ❌ O QUE NÃO FUNCIONOU

### 1. **Copy Grande Demais**
- ✗ Copys longas NÃO conectam em "fake persona"
- ✗ Audiência percebe como roteiro vs. história real
- **Aprendizado:** Scripts curtos, diretos, conversacionais
- **Novo padrão:** Max ~200-250 palavras por vídeo (testa em próximas)

### 2. **HeyGen Sync é Fraco**
- ✗ Gestual do personagem não acompanha sentimento da fala
- ✗ Exemplos: Momentos tristes/tensos → personagem continua sorrindo
- ✗ Disconnect visual = nega autenticidade
- **Aprendizado:** HeyGen não é ferramenta ideal para conteúdo emocional/story
- **Alternativa possível:** Considerar vídeo manual (gravação pessoal) OR outro provider

### 3. **Copy + Gestual Desalinhados**
- ✗ Quando copy é grande E gestual é genérico = falso
- **Padrão:** Se usar HeyGen, copy deve ser SUPER curta OU gestual deve ser customizado (não padrão)
